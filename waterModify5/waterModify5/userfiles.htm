一 概况

（一） 运行硬件环境

客户端最低配置：

F  CPU：Pentium4 2.4G；

F  内存：512M；

F  显存64M；

F  硬盘：40G；

服务器最低配置：

F  CPU：Pentium4 2.4G；

F  内存：512M；

F  硬盘：500G；

（二） 运行软件环境

客户端最低配置：

F  操作系统：MS Windows XP；

F  .NET运行环境：MS .NET Framework 4.0以上。

服务器最低配置：

F  操作系统：MS Windows XP Professional；

F  .NET运行环境：MS .NET Framework 4.0以上；

F  数据库：SQL Server2005；

F  IIS 5.0 以上。

（三） 编程语言及其版本

编程语言采用VS 2012编程平台下的WebService和Asp.net 4.0。

（四） 源程序行数

源程序共有10000多行。

（五） 主要功能及技术特点

本软件的主要功能及技术特点如下所示：

1.本软件针对城市排水系统的复杂性，实行“分片区”管理，实现对城市下水井盖的丢失情况和主要路段的下水井的液位实时监控。

2.本软件允许专家级别管理员对城市排水系统中的参数进行配置，添加、修改和查询相关的井盖模块信息。

3.本软件采用百度地图API，实现在地图上实时显示井盖的丢失情况和下水井的液位信息，并能够实现报警功能，并发送信息给相关人员，使其可以及时处理出现异常的模块。

4.本软件具有的规则生成工具可以将专家的经验转化规则，不断扩充知识库，提高系统的可靠性和灵活性。

（六） 用途

该网站实现城市排水系统的监控管理，实现对井盖丢失情况的监控，出现异常，及时报警。对重点位置的水井液位进行监控，但出现水井液位过高时，有报警提示，防止出现城市水倒灌影响城市交通等事故。

二 系统简介

（一） 概述 

城市排水系统管理网站(软件)由“导航栏”、“菜单栏”与多个子页面组成，其中“导航栏”能够提供快捷操作选项，根据不同交互需求和操作人员级别，会有不同的响应页面，以增强系统的灵活性和隐蔽性。

（二） 应用范围和对象

本系统软件应用于城市排水系统的管理，使用对象包括城市公共设施管理人员（本软件建议专家人员使用）、城市排水系统维护人员、城市管道维护人员等。

（三） 系统特色

该网站主要提供了监测数据发布，数据关联，数据管理，用户权限分配等功能。并能实现对井盖的实时监控和报警。

（五） 主要功能简介

该网站采用Zigbee模块发送信息，存储到数据库中，网站根据数据库存储的信息，将信息实时显示在网站上，实现井盖丢失报警、水井的液位信息显示与报警灯功能，实现对城市排水系统的监控。

该网站实现城市排水系统的监控管理，实现对井盖丢失情况的监控，出现异常，及时报警。对重点位置的水井液位进行监控，当出现水井液位过高时，有报警提示，防止出现城市水倒灌影响城市交通等事故。

3用户管理

具有管理员身份的用户，可以使用用户权限管理功能，可以添加、编辑和删除用户信息。登录网站以后，在左侧导航栏选择“权限管理”一项，会出现图17所示的用户列表信息。

       用户列表，有三个作用：

（1）           显示已有用户，查看添加操作是否成功；

（2）           查找并删除目标用户；

（3）           用于修改用户基本信息和为用户配置设备所有权，点击需要修改的用户行，可以编辑和删除用户信息，相关操作下面会继续介绍。

                                                                                                                                                                    

图17 城市排水管理系统用户列表

1）添加用户信息

在用户列表页面，单击“添加”按钮，弹出用户信息对话框（如图18所示），其中带有“！”的文本框为必填内容，否则程序会报错。点击“保存”就可以向数据库里添加用户，点击“取消”放弃添加用户信息。这里的用户是指城市公共设施管理人员。

                                                                                                                                                                                                                     

图18 城市排水管理系统添加用户信息

图19 城市排水管理系统编辑用户信息

单击需要删除的用户信息后，“删除”按钮被激活，点击“删除” ，弹出“删除用户信息”对话框（如图20所示），对井盖模块信息进行修改，点击“OK”完成修改，或者点击“Cancel”放弃删除。

（三） 系统小结

该网站主要提供了监测数据发布，数据关联，数据管理，用户权限分配等功能。通过Zigbee模块，实时发送井盖信息数据，能够实时监控井盖是否出现异常，对城市重点区域有水井的液位信息的监控模块，可以实时监控液位信息，防止出现水位过高，影响城市排水系统的正常运行。实现了对城市排水系统的实时监控。
